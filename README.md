This repository contains the source code for the "Fülesbagoly Vendégház," built with Next.js and TypeScript. The project utilizes Tailwind CSS for modern, responsive styling and integrates Sanity.io as a headless CMS to manage dynamic content efficiently. It also features NextAuth for handling authentication and secure routes within the application. To run the project locally, install the dependencies using npm install and start the development server with npm run dev.
